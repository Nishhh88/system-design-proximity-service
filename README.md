# system-design-proximity-service

This repository contains my system design notes for building a scalable location-based service similar to Google Maps, Uber nearby drivers, or Yelp nearby places.

## Problem Statement

Design a service that allows users to search for nearby places or users based on their current location.

Example use cases:

- Find restaurants near me
- Find nearby drivers
- Show stores within a 5 km radius
- Track live user location updates

## Functional Requirements

- Users can update their current location
- Users can search for nearby places/users

## Non-Functional Requirements

- Low latency for nearby search
- High availability
- Scalable for millions of users
- Efficient geospatial indexing
- Eventually consistent location updates are acceptable
- Secure user location data
